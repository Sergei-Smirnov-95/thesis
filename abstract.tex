\keywords{%
  диагностика, климатические параметры,
  программно-аппаратный комплекс,
  системы хранения данных
}

\abstractcontent{
 В работе изложен подход к диагностированию систем хранения данных основанный на онтологических моделях компоненов СХД. Кроме того, для обеспечения полноты собираемых метрик был разработан аппаратно-программного комплекса сбора климатических параметров для систем хранения данных. 

 В результате работы был создан действующий прототип программно-аппаратного комплекса сбора и отображения климатических параметров системы хранения данных.
 Достигнутые результаты используются в проекте лаборатории НИЛ АСПОД СПбПУ.
}

\keywordsen{climatic parameters,diagnostics,hardware and software system
}

\abstractcontenten{
The paper presents an approach to diagnosing data storage systems based on ontological models of component storage systems. In addition, to ensure the completeness of the collected metrics, a hardware-software complex for collecting climatic parameters for data storage systems was developed.

As a result of the work, a working prototype of a software and hardware complex for collecting and displaying climatic parameters of a data storage system was created.
The achieved results are used in the project of the laboratory NIL ASPOD SPbPU.
}
